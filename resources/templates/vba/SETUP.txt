HOW TO BUILD THE XLSM TEMPLATE
==============================

Goal: create resources/templates/items-import-base.xlsm
      (the PHP controller loads this, injects live units, and serves it as the download)

STEP 1 – Open Excel and download the current template
  - Go to the app's Import Items page and click "Download Template"
  - Open the downloaded items-import-template.xlsx in Excel

STEP 2 – Save as Macro-Enabled Workbook
  - File > Save As > choose "Excel Macro-Enabled Workbook (*.xlsm)"
  - Save anywhere temporarily (e.g. Desktop)

STEP 3 – Open the VBA Editor
  - Press Alt+F11

STEP 4 – Paste ThisWorkbook code
  - In the Project tree, double-click "ThisWorkbook"
  - Paste the contents of ThisWorkbook_code.bas

STEP 5 – Insert a standard module
  - Insert > Module
  - In the Properties window (F4) set Name = modItemForm
  - Paste the contents of modItemForm.bas

STEP 6 – Insert a UserForm
  - Insert > UserForm
  - In the Properties window set Name = frmAddItem
  - Set Caption = "Add Inventory Item"
  - Set Width = 570, Height = 660

STEP 7 – Add controls to the form
  Use the Toolbox (View > Toolbox) to add each control.
  Set the Name property (in Properties window) EXACTLY as listed.

STEP 8 – Paste the form code
  - Double-click the form canvas (not a control) to open its code window
  - Paste the entire contents of frmAddItem_code.bas

STEP 9 – Save the file
  - Save as "items-import-base.xlsm" in:
    resources/templates/items-import-base.xlsm
    (relative to project root: D:\Projects\mmc\lis-2\resources\templates\)

STEP 10 – Test
  - Re-download the template from the app
  - The downloaded file will be items-import-template.xlsm
  - Open it; the form should appear automatically
  - Fill in items; each "Add Item" click writes a row to the Items sheet
  - Upload the saved file on the Import Items page

NOTES
-----
- The _units sheet is repopulated by PHP on every download, so unit dropdowns
  in the form always reflect the current database contents.
- You can also upload the filled .xlsm directly (PHP now accepts it).
- If units list is empty, the unit dropdowns in the form will be empty too.
